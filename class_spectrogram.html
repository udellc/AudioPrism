Detailed Description

Spectrogram holds the frequency domain data over multiple time windows.

This class implements a circular buffer to store multiple windows of frequency domain data. Pushing new windows of data will overwrite the oldest window stored. It requires the buffer data to be allocated by the user.

Spectrogram::Spectrogram ( const uint16_t numWindows)

Creates a Spectrogram with a buffer to hold data.

Allocates a buffer to hold data for the specified number of windows, with the specified number of frequency bins in each window. Each window will contain a number of frequency bins equal half the WINDOW_SIZE. Initializes the current index to 0.

numWindowsThe number of time windows the Spectrogram holds.

void Spectrogram::pushWindow ( const float * data)

Pushes a new window to the Spectrogram.

Takes new window data and adds it to the circular buffer If the buffer index reaches the maximum column index, it wraps around and overwrites the oldest data, setting the index back to 0
